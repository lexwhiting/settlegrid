Both sg.wrap() and settlegridMiddleware() automatically support MCP, MPP, x402, AP2, Visa TAP, UCP, ACP, Mastercard Agent Pay, Circle Nanopayments, and REST — no protocol-specific code needed. SettleGrid detects the protocol from each incoming request and handles settlement automatically.
