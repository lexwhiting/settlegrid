## Quick Start

```bash
npm install
cp .env.example .env   # Add your SettleGrid API key
npm run dev
```

## Methods

| Method | Description | Cost |
|--------|-------------|------|
| `generate_name(...)` | Generate Name | 1¢ |
| `generate_username(...)` | Generate Username | 1¢ |

## Parameters

### generate_name
- `gender` (string; count?: number, optional)

### generate_username
- `style` (string; count?: number, optional)

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `SETTLEGRID_API_KEY` | Yes | Your SettleGrid API key |
